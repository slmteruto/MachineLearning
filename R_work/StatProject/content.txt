
1. 평균 차이 검정 (집단을 기준으로 하는가)
  - 집단간의 평균 차이를 검정
  - T-Test, ANOVA, ANCOVA, MANOVA
  (1) 사용경우
    - 명목변수 → 연속변수
  (2) 차이검정 방식
    1) t-test
      가. 집단이 1개일 때
        - One sample T-Test
        - 단일표본 평균검정
        ex. 하나의 집단에서 평균의 차이가 있는가 없는가
      나. 집단이 2개일 때
        - Independent sample T-Test
        - 독립표본 평균검정
        ex. 남자와 여자의 성적 차이가 있는가 없는가
      다. 시간에 따른 차이
        - Paired sample T-Test
        - 쌍체 평균검정
        
        
# ANOVA 방식

# 집단이 3개 이상일 때 : One Way ANOVA  (일원 배치 분산 분석)

#        - Independent T-test의 확장 

# 시간의 따른 차이 검정 : Repeated Measured ANOVA (반복 측정 분산 분석)

#         밥먹기 전 후 차이가 있냐 두가지 경우밖에 없으므로 - paired

#         여러개 시간이 있다면 이럴 때 ANOVA를 쓴다. 

# 요인에 따른 차이 검정 : Two Way ANOVA (이원 배치 분산 ??)

#         1000도, 1500도 치킨 맛 차이 이럴 경우 Independent Test를 하면 되는데

#         1000도, 1500도 치킨 맛 차이 , 오븐, 기름에 튀겼을 때 맛차이 일 경우엔 t-test를 두번 하는게 아니고 two-way를 쓴다 

# 시간 + 요인에 따른 차이 : Two Way Repeated Mesured ANOVA (?)

#         집단에서 통증에 대해 실험군, 대조군일땐 t-test 쓰면 되긴 하다.

#         하지만 실험군, 대조군이 늘어나면  One way ANOVA로 갈아타면 되는데

#         만약 시점까지 존재하게 된다면 이걸 써야한다.  실험전, 실험후 + 실험군 대조군




2. 관계 검정

  - 변수 기준
  - 연속 변수 -> 연속변수
  ---------------------------------------------------
  
  (1) 상관 분석 - 관계가있냐 없냐만 판단
    - 1:1로 서로 동등한 입장에서 관계를 분석
    - 두 변수는 연속형 변수이어야 한다.
    - 선형 함수이어야한다
    - 상관계수(r계수) : 1 ~ -1, 설명계수(r^2) : 0.5^2 = 0.25
    - 피어슨의 상관 계수 (모수적인 방식), 스피어만 상관계수(비모수적), kendall의 tau(비모수적)
    
    
  (2) 회귀 분석 - 상관관계에 더 나아가서 추정판단이 가능해짐
    - 인과관계  (상관분석에선 인과관계를 함부로 해석하면 안됐다)
    - 1(독립변수 또는 원인변수):1(종속변수, 결과변수) -> 단일 회귀분석
    - 다:1 -> 다중 회귀분석   (회귀분석의 일반적인 형태이다.)
    - 조건이 있다.
      - 연속변수여야함.(또한 선형성도 이루어야한다./직선형태)
      - 정규분포 (데이터가 정규분포여야함)  - 정규분포이기만 하면 데이터는 절반이상 성공 
      - 등분산이어야 함. 
      - 다중공선성이 작아야함 - 다중 공선성이란 독립성을 이야기. (각 컬럼끼리 비슷한놈이 적어야 된다는 것)
      
    - 일반 선형 모델(Ordinary Linear Model)   - 최소 제곱법 (직선)
      
    - 일반화 선형 모델 (General Linear Model) - 로지스틱 회귀분석 (시그모이드 sigmoid함수), 포아송 회귀분석 
        포아송 : 자주 일어나지 않은 상황에 쓰이는..

    - 질적변수 -> 질적변수
